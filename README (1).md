### 🔒 MOCK DRILL — Automated Security Audit Platform @ ISRO
> **Production system built at Space Applications Centre, ISRO**

Automated compliance audit platform for critical network infrastructure. Consolidated 69+ security checks across DNS, email, web, and SSH/FTP into a single scan engine.

- **Impact:** Reduced 4-day manual audit to 4 hours (90% effort reduction)
- **Stack:** FastAPI · Celery · Redis · PostgreSQL · React · TypeScript · Docker · Nginx
- **Features:** CVSS v3.1 scoring · Auto-generated remediation playbooks · RBAC (4 roles) · Real-time dashboard · One-click PDF reports
